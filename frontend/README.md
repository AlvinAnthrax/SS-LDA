# SS-LDA Frontend

## Setup

1. Install dependencies:

   ```bash
   cd frontend
   npm install
   ```

2. Run development server:

   ```bash
   npm run dev
   ```

3. Open browser:
   - biasanya `http://localhost:5173`

## Notes

- Pastikan backend FastAPI berjalan di `http://localhost:8000`
- Endpoint yang dipanggil: `POST /analyze`
